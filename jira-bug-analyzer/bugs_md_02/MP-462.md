# MP-462: API - Datasource - misleading protocol usage

**Status:** Backlog

**Assignee:** Unassigned

**Description:**
The sync_schema endpoint uses a GET request, but it performs a schema synchronization, which is a state-changing operation and should use POST or PUT instead.
