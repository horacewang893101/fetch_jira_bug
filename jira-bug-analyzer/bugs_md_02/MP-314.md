# MP-314: No onboarding for new users

**Status:** Backlog

**Assignee:** Unassigned

**Description:**
*Location:* Entire application  
*Severity:* High  
*Impact:* New users don’t know where to start.

*Reproduction Steps:*

# Log in as a new user
# No guide, tooltips, or tutorial is shown

*Evidence:* 

* Screenshot: [dashboard-no-onboarding.png]

*Recommendation:* 

* Add a visible “Getting Started” guide for new users
* Add prominent suggestions/quick actions on the home page (e.g., “Create Agent”, “Open Chat”, “Browse Agents”)
* Implement an interactive in-app tour
* Add explanatory tooltips near key elements
