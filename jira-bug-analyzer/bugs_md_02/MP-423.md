# MP-423: Azure Key Vault logs not collected

**Status:** Backlog

**Assignee:** Unassigned

**Description:**
* *Affected:* Azure Key Vault
* *Description:* Key Vault logging is not enabled/collected centrally.
* *Impact:* Access failures, permission issues, and audit trails are difficult to investigate.
* *Recommended fix (what/where):*
** Enable Key Vault *Diagnostic settings* to forward audit/access logs to the central log destination.
