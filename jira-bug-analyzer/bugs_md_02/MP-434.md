# MP-434: Unclear error when trying to upload the same file twice

**Status:** Backlog

**Assignee:** Unassigned

**Description:**
*Location:* Knowledge → File upload  
*Severity:* High  
*Impact:* User doesn't understand why the upload failed - no clear message that a file with this name already exists.

*Reproduction Steps:*

# Go to Dashboard → Knowledge → Edit Org Workspace
# Upload a file
# Try to upload the same file again
# Observe the "Upload Failed" error without a clear explanation of the cause

*Evidence:* Screenshot:

!image-20260116-131853.png|width=454,alt="image-20260116-131853.png"!

*Recommendation:*

* Display a clear message: "A file with this name already exists in this directory."
* Offer options: overwrite, rename, cancel
* Improve the error message to be more user-friendly

*Affected Screens:*

* Knowledge → File upload
