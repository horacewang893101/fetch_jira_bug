# MP-405: Application settings provisioned from terraform code as single KV secret

**Status:** Backlog

**Assignee:** Unassigned

**Description:**
Whole appsettings provisioned as big key vault secrets which prevents more granular secrets management such as db password rotation. The secret values are part of the [vault.tf|http://vault.tf] template either hardcoded in the file or with some parts passed as a variables.
