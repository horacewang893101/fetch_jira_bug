# MP-485: No Persistent Volume Backup Strategy

**Status:** Backlog

**Assignee:** Unassigned

**Description:**
*Issue*: Azure File Shares (robot-data-pvc, knowledge-pvc, fonts-pvc) have no backup policies configured.

*Risks*: 

* Permanent loss of application data, knowledge base, uploaded files
*  no recovery from accidental deletion or corruption.

*Fix*: Enable Azure Backup for File Shares;
