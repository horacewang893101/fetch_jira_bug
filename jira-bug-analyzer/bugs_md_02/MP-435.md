Executive Summary

An in-depth performance analysis of the AI Central application was conducted using Lighthouse and Core Web Vitals metrics. The application exhibits *critical performance issues* across all tested pages, with scores ranging from *33/100 to 45/100*. Key concerns include extremely high Largest Contentful Paint (LCP) times exceeding 20 seconds, significant Total Blocking Time (TBT), and large JavaScript bundle sizes impacting initial load times.

A total of 8 issues related to application performance were identified, including *6 critical (P0)* and *2 high (P1)*.
