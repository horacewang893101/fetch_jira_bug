# MP-482: PostgreSQL Geo-Redundant Backup Disabled

**Status:** Backlog

**Assignee:** Unassigned

**Description:**
*Issue*

*  geo_redundant_backup_enabled = false in terraform state;
* backups limited to 7 days retention in single region.

*Risks:*

* Cannot recover from regional disasters;
* limited point-in-time recovery window;
* permanent data loss if region fails.

*Fix:*

* Enable in [pgsql.tf|http://pgsql.tf]: geo_redundant_backup_enabled = true 

* increase backup_retention_days = 35 (max supported).
