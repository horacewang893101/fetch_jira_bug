# MP-484: Storage Account Using LRS Replication

**Status:** Backlog

**Assignee:** Unassigned

**Description:**
*Issue*: Azure Storage uses {{account_replication_type = "LRS"}} ([nas.tf:19|http://nas.tf:19]) - Local Redundant Storage only.

*Risks*:

* Data loss if datacenter fails;
* no geographic redundancy for persistent volumes; 

*Fix*: Change to ZRS (Zone-Redundant) or GRS (Geo-Redundant): {{account_replication_type = "ZRS"}} for zone protection or {{"GRS"}} for regional disaster recovery.
