# MP-276: Auto Table Creation Enabled for Logging in serilogsettings secret

**Status:** Backlog

**Assignee:** Unassigned

**Description:**
*Description*
{{needAutoCreateTable=true}} allows Serilog to create tables automatically in the database. This requires elevated permissions and increases risk if exploited.

*Impact*
Attackers could abuse this feature to create arbitrary tables or modify schema if they gain access to the logging sink.

*Evidence*
{{"needAutoCreateTable": true}}

*Recommended Fix*

* Disable auto table creation in production.
* Pre-create logging tables with strict permissions (INSERT-only for logging account).
