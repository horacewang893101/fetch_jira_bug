# MP-265: PostgreSQL Uses Default postgres Superuser

**Status:** Backlog

**Assignee:** Unassigned

**Description:**
The application connects using the default {{postgres}} superuser account and a default password, granting unrestricted privileges. The database is hidden behind a private link, but using default credentials creates security risk nonetheless.

*Screenshot:*

!image-20260105-114130.png|width=891,alt="image-20260105-114130.png"!



*Recommended Fix:*

* Disable or restrict {{postgres}} login for applications.
* Create *least-privileged* app role per database.
* Rotate credentials; prefer *Azure AD auth* with Managed Identity if supported.
