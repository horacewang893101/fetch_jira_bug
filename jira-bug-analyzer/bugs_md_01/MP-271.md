# MP-271: Hardcoded LangFlow Admin Credentials in Knowledge workload settings

**Status:** Backlog

**Assignee:** Unassigned

**Description:**
*Description*
Admin username/password ({{LANGFLOW_SUPERUSER}}, {{LANGFLOW_SUPERUSER_PASSWORD}}) are defined in plaintext, enabling direct administrative login.

*Impact*
Immediate admin takeover of LangFlow UI/API, privilege escalation, and configuration tampering.

*Evidence (masked)*

* {{LANGFLOW_SUPERUSER=admin}}
* {{LANGFLOW_SUPERUSER_PASSWORD='********'}}
* {{LANGFLOW_AUTO_LOGIN=false}} (doesnâ€™t mitigate plaintext admin cred exposure)



*Recommended Fix*
Remove static admin credentials; use *Azure AD / OAuth* for admin login with MFA. If local admin is required, rotate to a strong random password and store as a separate Key Vault secret.
