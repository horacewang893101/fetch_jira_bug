# MP-357: Missing Focus Visible on Interactive Elements

**Status:** Backlog

**Assignee:** Unassigned

**Description:**
*Priority*: Critical  
*Impact*: Accessibility (WCAG 2.1 AA - 2.4.7), Keyboard Navigation  
*Category*: Keyboard Navigation

*Problem*:

* Many buttons and links do not show a visible focus indicator
* This makes keyboard navigation difficult/impossible
* Violation of WCAG 2.1 Success Criterion 2.4.7 (Focus Visible)

*Affected Elements*:

* Agent cards on the home page
* Category filter buttons
* Sidebar menu items
* Feature cards (Deep Research, AI Search, etc.)

*Recommended Fix*:

{code:css}*:focus-visible {
  outline: 2px solid #0066cc;
  outline-offset: 2px;
}{code}
