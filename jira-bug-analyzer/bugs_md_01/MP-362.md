# MP-362: Missing “Skip to Content” Link

**Status:** Backlog

**Assignee:** Unassigned

**Description:**
*Priority*: Medium  
*Impact*: Accessibility (WCAG 2.4.1), Keyboard Navigation  
*Category*: Bypass Blocks

*Problem*:

* There is no “Skip to main content” link for screen reader users
* Keyboard users must tab through the entire navigation every time

*Recommended Fix*:

{code:html}<a href="#main-content" class="skip-link">Skip to main content</a>{code}
