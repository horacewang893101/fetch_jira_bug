# MP-268: Static Salts and Encryption Keys hardcoded in Config in Knowledge workload settings

**Status:** Backlog

**Assignee:** Unassigned

**Description:**
*Description*
Static salts and symmetric encryption keys are hardcoded in {{appsettings}} config file stored in Key Vault, weakening password hashing and enabling decryption if data is stolen.

*Impact*
Facilitates offline cracking and data decryption upon breach.

*Evidence*
{{SaltSecrets=[...]}}
{{DataSourceEncryptKey=...}}
{{EncryptConfig.DesEncryptKey=...}}, {{DesEncryptIV=...}}



*Recommended Fix*

* Store keys in *Key Vault*; rotate periodically.
* Use *per-user/per-record salts* with strong KDF (Argon2id).
* Replace DES with *AES-GCM* or equivalent AEAD.
