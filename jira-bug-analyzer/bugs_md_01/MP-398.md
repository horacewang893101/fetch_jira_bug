# MP-398: [Recommendation] MCPM service - Add memory limit

**Status:** Backlog

**Assignee:** Unassigned

**Description:**
Requests are low (*100m / 500 Mi*) but *no memory limit* is set.

*Impact:*
Risk of unbounded memory growth (leaks/spikes) affecting node stability.

*Recommended Fix:*

* *CPU request:* 20–50m
* *Memory request:* 128–256 Mi
* *Memory limit:* 512 Mi–1 GiB.
