# MP-267: Swagger/OpenAPI Exposed in Production

**Status:** Backlog

**Assignee:** Unassigned

**Description:**
Swagger UI and OpenAPI JSON are exposed, enabling adversaries to enumerate endpoints and payload schemas. No authentication is required to access the swagger UI.



*Impact*
Accelerates exploit development and API misuse.



*Screenshot:*

!image-20260105-114958.png|width=939,alt="image-20260105-114958.png"!

[trial.aicentral.top/webapi/openapi.json|https://trial.aicentral.top/webapi/openapi.json]



*Recommended Fix*

* Disable Swagger UI/JSON in production or restrict behind IP allowlist/admin auth.
* Provide internal-only documentation endpoints.
