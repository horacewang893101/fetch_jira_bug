# MP-363: Missing Role Landmarks (Page Semantics)

**Status:** Backlog

**Assignee:** Unassigned

**Description:**
*Priority*: Medium  
*Impact*: Accessibility, Screen Reader Navigation  
*Category*: Semantic HTML

*Problem*:

* The page lacks clear semantic landmarks that support navigation (header/nav/main/footer)

*Recommended Fix*:

{code:html}<header role="banner">...</header>
<nav role="navigation">...</nav>
<main role="main" id="main-content">...</main>
<footer role="contentinfo">...</footer>{code}
